Audited logic

Scoring assumptions

  1. Group order: exact 1st/2nd/3rd/4th position earns 8/6/4/2 points.
  2. Perfect group: all four positions correct adds a 10-point bonus.
  3. Best third-place teams: each correct qualifier earns 3 points.
  4. Knockout scoring: the current Sheet bracket has picks and actual results but no scoring formulas, so bracket accuracy is tracked separately until round values are added.
  5. Ties: the Sheet uses RANK.EQ on points only. The app prefers official live table rank, including tournament tie-breakers.
